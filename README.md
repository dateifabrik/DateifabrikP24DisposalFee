# DateifabrikP24DisposalFee

Plugin zur Berechung der Verpackungskosten für Packing24

***

- Basket:
    - Anzahl Artikel
    - Gewicht des jeweiligen Artikels (NOT NULL)
    - Material (NOT NULL)

- Berechnung 1:
    - [Anzahl Artikel] x [Gewicht des jeweiligen Artikels] x Kosten für dieses [Material]
